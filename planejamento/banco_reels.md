# Banco de Reels — ZocLabs (4 semanas)

Cadência: **2 reels novos de tech/semana** + **2 shorts da Copa/semana** (já saem da routine Inteligência Artilheira → não re‑roteirizados aqui).

Molde fixo (tendência Reels 2026 — estrutura > produção):
- **0–1s:** pattern interrupt (movimento/surpresa) + 1ª linha falada + texto na tela.
- **1–3s:** curiosidade/promessa (por que ficar).
- **miolo:** o conteúdo (demo de tela ou câmera).
- **final:** 1 linha compartilhável + CTA ("envia pra…" / "salva").
- Texto na tela ≤ 2 linhas, 3–5 palavras/linha, legenda sempre (maioria assiste sem som).

Duração alvo: 20–35s (ganchos curtos podem ir a 15s).

## Semana 1

### R1 — "Claude comum x Claude Code em 30s" · câmera + 1 print · fácil · 25s
- **Gancho 3s:** "Você usa o Claude errado pra trabalho — e eu também usava."
- **Roteiro:** (câmera) "O Claude do navegador conversa. O Claude Code AGE: ele lê seus arquivos, edita e roda comando." → (corta pro print) "Olha ele revisando um projeto inteiro sozinho." → (câmera) "Pra estudar, o do chat. Pra trabalhar num projeto, o Code."
- **Cena:** você falando + 1 corte de tela mostrando o Claude Code rodando.
- **Texto na tela:** "Chat = conversa" / "Code = AGE no projeto".
- **CTA:** "Envia pro dev que só usa o chat. Salva."
- **Vira:** corte do carrossel #7 e do V3.

### R2 — "O fluxo de Power Automate que economiza 2h/semana" · tela · médio · 30s
- **Gancho 3s:** "Esse fluxo me devolve 2 horas por semana. Monta em 4 passos."
- **Roteiro:** (tela) gatilho 'quando chegar e‑mail' → condição 'tem anexo' → 'salvar no SharePoint' → 'avisar no Teams'. Fala por cima, rápido.
- **Cena:** gravação de tela do Power Automate (acelerada 1.5x).
- **Texto na tela:** "E‑mail → pasta, sozinho" / âmbar: "2h/semana".
- **CTA:** "Salva pra montar. Doc em PT na descrição."
- **Vira:** corte do carrossel #14 e do V1.
